## 🧠 About the Project / درباره برنامه

<details>
<summary>English</summary>

**AmirCollider App** is a fully animated, dual-language Android application built in Unity.  
It features smooth UI transitions, a custom loading system, local/global server fallback, and dynamic content refresh.

</details>

<details>
<summary>فارسی</summary>

**AmirCollider App** یک برنامه اندرویدی دو زبانه است که با Unity ساخته شده.  
دارای انیمیشن‌های روان، رابط کاربری جذاب، سیستم لودینگ حرفه‌ای، پشتیبانی از سرور محلی/گلوبال و به‌روزرسانی محتوای پویاست.

</details>
